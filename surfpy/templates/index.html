HawaiiClimate

Hawaii Climate API

HawwaiClimate Flask APP allows you access basic information that help to do climate analysis on Hawaii area.

Requests

Climate observations

Returns the date and the climate observations form the last year. All calls for the observations are in this format:
http://127.0.0.1:5000/api/v1.0/[components]

Components

Components of the url describe what type of data you would like. They are separated by forward slashes.

Name Description Sample Values
Climates observations* Return the dates and the climate observations of the last year. Valid values: "pobs", "tobs"
Weather stations* Return the weather stations of the dataset. stations
*required

Examples of usage

/api/v1.0/pobs
Returns the dates and the last year temperature observations.

Daily Normals

Return a JSON list of the minimum, the average, and the maximum (normals) of climatic observations for a given start or start-end range. When given the start only, return normals calculated for the observations since the start date. When given the start date and the end date, return normals calculated for the observations between the start date and end date inclusive. All calls for the observations are in this format:
/api/v1.0/normals/[components]/[start]/[end]

Components

Components of the url describe what type of data you would like. They are separated by forward slashes.

Name Description Sample Values
Climates observations* Return the daily normals gor given `start` and `end` range. Valid values: "pobs", "tobs".
Valid `start` and `end` formats: "yyyy-mm-dd"/
*required

Examples of usage

/api/v1.0/normals/tobs/2016-10-01
Returns the normals for temperature observations since 2016-10-01.
/api/v1.0/normals/tobs/2016-10-01/2017-12-05
Returns the normals for temperature observations collected between 2016-10-01 and 2016-12-05 inclusive.
